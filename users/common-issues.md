## üîê Login Issues

### Cannot Login / Wrong Password
**Solution:**
1. Click "Forgot Password?" on login page
2. Enter your email
3. Check inbox for reset link
4. Create new password (minimum 12 characters)

### Email Not Verified
**Error:** "Email not verified"
**Solution:**
1. Check your email inbox for verification email
2. Click the verification link
3. If no email, click "Resend verification link" on login page
4. Wait a few minutes and check spam folder

## üìÑ Document Upload Issues

### File Too Large
**Error:** "File size exceeds limit"
**Solution:**
- Maximum file size: 5MB
- Compress images before uploading
- For PDFs, use PDF compressor tools
- Split large files into smaller parts if necessary

### Wrong File Format
**Error:** "Invalid file type"
**Solution:**
- Accepted formats: JPEG, PNG, WebP, PDF
- Convert other formats using online converters
- Ensure file extension is correct (.jpg, .png, .pdf)

## üéì Enrollment Issues

### Cannot Enroll in Course
**Possible Reasons:**
- Section is full (at capacity)
- Prerequisites not met
- Not eligible for that year level
- Enrollment period closed
- Outstanding balance

**Solution:** Contact registrar office

### Course Not Showing in List
**Reasons:**
- Course not offered this semester
- Section not created yet
- Already enrolled in maximum units
- Course conflicts with existing schedule

## üîß Technical Issues

### Page Not Loading
**Try:**
1. Refresh the page (Ctrl+F5 or Cmd+Shift+R)
2. Clear browser cache
3. Try different browser (Chrome, Firefox, Edge)
4. Check internet connection
5. Try incognito/private mode

### Session Expired
**What happened:**
- You were inactive for 15 minutes
- Session automatically logged out for security

**Solution:**
1. Login again
2. You'll be redirected to your last page
3. Stay active or extend session when prompted
