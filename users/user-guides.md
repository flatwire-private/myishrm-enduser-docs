### Staff & Faculty Roles

#### [Admission Officer](admission-officer.md)
**Primary Responsibilities:**
- Review new applicant applications
- Verify submitted documents
- Approve applications and generate invoices
- Monitor applicant payment status

**Quick Actions:**
- Review pending applications
- Generate invoices for approved applicants
- View all invoices

---

#### [Accounting Staff](accounting-staff.md)
**Primary Responsibilities:**
- Verify payment proofs uploaded by applicants
- Approve or reject payments
- Generate financial reports
- Track student account balances

**Quick Actions:**
- View pending payment proofs
- Verify payment receipts
- Review financial transactions

---

#### [Registrar](registrar.md)
**Primary Responsibilities:**
- Convert approved and paid applicants to students
- Manage student enrollments
- Issue student IDs
- Process document requests
- Generate official records

**Quick Actions:**
- Convert applicants to students
- Manage student enrollments
- Issue academic documents

---

#### [Dean](dean.md)
**Primary Responsibilities:**
- Manage college courses and curriculum
- Create and edit course subjects
- Assign faculty to course sections
- Set class schedules
- Monitor college performance

**Quick Actions:**
- Create new courses
- Add subjects to curriculum
- Create course sections
- Assign faculty to sections

---

#### [Faculty](faculty.md)
**Primary Responsibilities:**
- Submit student grades
- Manage assigned classes
- Mark attendance
- View class rosters

**Quick Actions:**
- View assigned classes
- Submit grades
- Mark attendance

---

### Student Roles

#### [Student](student.md)
**Primary Responsibilities:**
- Enroll in course sections each semester
- View grades and academic records
- Pay tuition and fees
- Request official documents
- View class schedules

**Quick Actions:**
- Enroll in courses
- View grades
- Pay fees
- View schedule

---

#### [Applicant](applicant.md)
**Primary Responsibilities:**
- Complete application form
- Upload required documents
- Pay admission fees
- Upload payment proof
- Track application status

**Quick Actions:**
- Submit application
- Upload documents
- Upload payment proof
- View invoice
